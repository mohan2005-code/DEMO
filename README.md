# AI-Powered Debt Relief & Financial Recovery Platform

## 📌 Project Overview

The **AI-Powered Debt Relief & Financial Recovery Platform** is a web-based application designed to assist individuals in managing debt and making informed financial decisions. The platform leverages **Google Gemini AI** to provide intelligent financial guidance, debt analysis, settlement predictions, and automated negotiation support.

This project was developed as part of the **SmartBridge Academic Project**.

---

## 🚀 Features

- 🔐 User Authentication
- 🤖 AI-Powered Debt Analysis
- 💰 Loan Management
- 📊 Debt Settlement Prediction
- 📧 AI Negotiation Email Generator
- ⚖️ Know Your Rights Module
- 👨‍💼 Advisor Dashboard
- 👨‍💻 Admin Dashboard
- 📈 Financial Insights
- 📜 User History & Activity Logs

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/mohan2005-code/DEMO
```

### Backend Setup

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
